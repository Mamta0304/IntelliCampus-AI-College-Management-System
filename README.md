# IntelliCampus: AI-Powered College Management System with Hybrid RAG

## Overview

IntelliCampus is a full-stack academic management platform designed to streamline campus operations through centralized scheduling, attendance tracking, assignments, notifications, and AI-powered academic assistance.

The system integrates a Hybrid Retrieval-Augmented Generation (RAG) architecture that combines structured database retrieval with semantic document search to provide intelligent responses to academic queries.

## Key Features

* Role-Based Access Control
* Student Dashboard
* Faculty Dashboard
* Admin Dashboard
* Attendance Management
* Assignment Tracking
* Timetable Management
* Smart Notifications
* Academic Calendar
* AI Academic Assistant
* Hybrid RAG Pipeline
* PDF Document Processing
* Semantic Search using FAISS

## System Architecture

User Query
→ Document Retrieval (FAISS)
→ Database Retrieval
→ Hybrid RAG Processing
→ Llama 3.3 Response Generation
→ Intelligent Answer
